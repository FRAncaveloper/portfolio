# Currículo Interativo | Pedro Henrique de Oliveira Franco

## 💻 Sobre o Projeto

Este aplicativo foi desenvolvido inteiramente em **Flutter** e **Dart** com o objetivo de apresentar meu currículo de forma dinâmica e interativa. A aplicação transforma o currículo tradicional em uma experiência digital, demonstrando na prática minhas habilidades em desenvolvimento mobile, criação de interfaces responsivas e aplicação de conceitos de UI/UX.

A interface gráfica foi idealizada com uma temática "Pixel Art / Retrô", utilizando componentes estilizados sob medida, paletas de alto contraste e tipografia personalizada. Essa escolha de design teve como propósito demonstrar a flexibilidade na manipulação avançada de temas (`ThemeData`) e construção de widgets customizados (`Container`, `BoxDecoration`, `BoxShadow`), aplicando uma identidade visual própria e desvinculada dos componentes padrões do Material Design.

## 🛠️ Tecnologias Utilizadas

- **Dart:** Linguagem base da aplicação, utilizando conceitos de Programação Orientada a Objetos.
- **Flutter:** Framework principal utilizado para a construção da interface e implementação de rotas/navegação fluida entre telas (`Navigator`).
- **Google Fonts:** Integração de pacotes externos para utilização da tipografia `VT323`, compondo a identidade visual do projeto.
- **Git & GitHub:** Utilizados para o versionamento de código e versionamento do projeto.

## 📱 Estrutura da Aplicação

A arquitetura de navegação foi desenhada de forma linear e progressiva, conduzindo o recrutador ou usuário pelas seguintes seções:

1. **Apresentação:** Ponto de entrada com a declaração do perfil e objetivo profissional.
2. **Resumo e Educação:** Detalhamento do background acadêmico e interesses na área de tecnologia.
3. **Experiência Profissional:** Exposição das vivências práticas, com foco em comunicação, resolução de problemas e atendimento de demandas.
4. **Projetos Relevantes:** Demonstração de aplicações Flutter e prototipações (Figma/Photoshop) recentes.
5. **Status de Habilidades:** Tela final consolidando Hard Skills (Linguagens, Frameworks, Banco de Dados), Soft Skills e ferramentas de domínio.

## 🚀 Instalação e Execução

Para executar este projeto localmente, é necessário ter o ambiente do [Flutter configurado](https://docs.flutter.dev/get-started/install) em sua máquina.
1. Clone este repositório:
```bash
git clone https://github.com/FRAncaveloper/nome-do-seu-repositorio.git
```
2. Acesse o diretório do projeto:
```bash
cd nome-do-seu-repositorio
```
3. Instale as dependências necessárias:
```bash
flutter pub get
```
4. Execute a aplicação (recomendável a utilização de um emulador Android ou dispositivo físico):
```bash
flutter run
```
